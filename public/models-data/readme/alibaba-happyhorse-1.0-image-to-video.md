## Alibaba HappyHorse 1.0 Image-to-Video

**Alibaba HappyHorse 1.0 Image-to-Video** animates a single first-frame image into a short video clip, with optional prompt guidance, two output resolutions, and support for durations from 3 to 15 seconds.

## What makes it stand out?

- **Simple first-frame workflow:** Start from one image URL and turn it into motion.
- **Prompt-guided animation:** Add a prompt when you want more control over action, style, or atmosphere.
- **Two output resolutions:** Generate at `720P` or `1080P`.
- **Practical input constraints:** Works with common image formats up to 10 MB.

## Designed For

- Teams turning still artwork, concept frames, or product renders into motion clips.
- Creators producing short animated shots from posters, illustrations, or key art.
- Storyboard and previsualization workflows that need a lightweight first-frame animation path.

## How to Use

1. Provide an image URL to use as the first frame.
2. Add a prompt if you want to steer motion, pacing, or scene detail.
3. Choose `720P` or `1080P` based on your quality target.
4. Set a duration between 3 and 15 seconds.
5. Use a fixed seed when you want a more repeatable result.
