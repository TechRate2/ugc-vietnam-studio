# Flux-Dev-Lora

flux-dev-lora is a 12 billion parameter rectified flow transformer capable of generating images from textual descriptions. For more information, please read our blog post.

## Key Features

1.  Competitive prompt following: Matches the performance of closed-source alternatives in understanding and executing prompts.
2.  Efficient training with guidance distillation: This approach makes the model more efficient and responsive.
3.  Open weights: Empowering new scientific research and enabling artists to develop innovative workflows.
4.  Versatile usage: Outputs can be utilized for personal, scientific, and commercial purposes.

## Limitations

*   The model is not designed to provide factual or verified information.
*   Being a statistical model, it may amplify existing societal biases.
*   Generated outputs might not always perfectly correspond to the given prompts.
*   Prompt interpretation is strongly influenced by the phrasing of the input.
