# Google Veo 3.1 Lite — Image-to-Video

**Veo 3.1 Lite Preview** ([`veo-3.1-lite-generate-preview`](https://ai.google.dev/gemini-api/docs/models/veo-3.1-lite-generate-preview)) is a high-efficiency, developer-oriented video model from Google, optimized for **scalable, programmable** image-driven workflows. It produces **video with audio** and is positioned as a **cost-effective** option for teams building **high-volume** video applications. According to the [official model documentation](https://ai.google.dev/gemini-api/docs/models/veo-3.1-lite-generate-preview), **Veo 3.1 Lite Preview does not support 4K outputs or Extension**; consult the [Video generation](https://ai.google.dev/gemini-api/docs/video) docs for end-to-end features and limits.

---

## Overview

Generate videos by **animating an input image** using Google's **Veo 3.1 Lite** model.

A **lightweight, cost-effective** variant of Veo 3.1 that **always generates video with audio**.

---

## Supported options

The model supports:

- **Input images** up to **8MB** in size
- **720p** or **1080p** output resolution
- **4, 6, or 8** second duration (**1080p requires 8s**)
- **16:9** or **9:16** aspect ratio
- **Audio is always generated**
- **Safety filters** are applied to **both input images and generated content**
