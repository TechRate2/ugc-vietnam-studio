## Alibaba WAN 2.7 Video Edit

**Alibaba WAN 2.7 Video Edit** edits existing videos using text instructions, reference images, and style transfer, with support for multi-modal inputs.

## What makes it stand out?

- **Instruction-based editing:** Describe the desired changes in natural language, and the model applies them to the video.
- **Reference-guided editing:** Provide reference images to guide the style, appearance, or content of the edit.
- **Video style transfer:** Transfer the visual style of a reference image onto the source video while preserving motion.
- **Audio-aware output:** Generates videos with audio that follows the edited content (voiced or silent, depending on the input).

## Designed For

- Creators who need to modify existing video footage without manual frame-by-frame editing.
- Teams applying consistent visual styles across a set of video clips.
- Anyone who wants to experiment with video transformations using text prompts and reference images.

## How to Use

1. Provide the `video` URL of the clip you want to edit.
2. Write a `prompt` describing the changes (e.g., "change the character's outfit to a red dress, keep the reference image's hat").
3. Optionally add an `image` or `images` for style or content reference.
4. Set `duration` to clip the output length, or omit it to match the input video duration.
5. Choose the output resolution and aspect ratio.
