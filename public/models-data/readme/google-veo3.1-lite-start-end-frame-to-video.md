# Google Veo 3.1 Lite — Start–End Frame to Video

**Veo 3.1 Lite Preview** ([`veo-3.1-lite-generate-preview`](https://ai.google.dev/gemini-api/docs/models/veo-3.1-lite-generate-preview)) is Google's **high-efficiency** video model for creators and enterprises that need a **programmable**, **scalable** path from still frames to motion. It inherits **Veo 3.1–class** fidelity and control while emphasizing **developer-first** integration and **competitive pricing** at scale. Output is **video with audio**. The [model page](https://ai.google.dev/gemini-api/docs/models/veo-3.1-lite-generate-preview) notes that **Veo 3.1 Lite Preview does not support 4K outputs or Extension**; see [Video generation](https://ai.google.dev/gemini-api/docs/video) for broader capability coverage.

---

## Overview

Generate videos by **animating between a first and last frame** using Google's **Veo 3.1 Lite** model.

A **lightweight, cost-effective** variant of Veo 3.1 that **always generates video with audio**.

---

## Supported options

The model supports:

- **Input images** up to **8MB** in size
- **720p** or **1080p** output resolution
- **8** second duration
- **16:9** or **9:16** aspect ratio
- **Audio is always generated**
- **Safety filters** are applied to **both input images and generated content**
